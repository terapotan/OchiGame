# ビルドする際の注意点
このリポジトリには、著作権の関係上素材ファイルが含まれていません。(キャラクター画像、音楽ファイルなど)そのため、このプロジェクトをクローンしてビルドを行っても**正常に動作しません**。どのようなコードが書かれているのか参考にする目的でご使用ください。
